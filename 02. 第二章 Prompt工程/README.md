## 核心概念

**Prompt = 任务定义 + 上下文 + 约束条件**

好的 Prompt 具备三个特征：**清晰（Clear）**、**具体（Specific）**、**结构化（Structured）**

## 章节速览

- **第一节**：理解 AI 处理 Prompt 的机制——概率预测、Token、Attention
- **第二节**：掌握核心工程技术——Zero-shot / Few-shot / CoT / ToT / Skeleton / Chaining
- **第三节**：学习如何结构化设计 Prompt——模块化 + 位置意识 + 渐进式注入
- **第四节**：针对不同任务类型的最优写法——代码 / 分析 / 写作 / 推理
