## 核心概念

**上下文 = 系统提示 + 对话历史 + 当前输入 + 外部上下文**

上下文工程的核心问题：**在有限窗口中，如何提供最相关、最高质量的信息**

## 章节速览

- **第一节**：理解上下文是什么——四部分组成、Attention 机制、位置效应
- **第二节**：上下文窗口管理——信息优先级、压缩技术、滑动窗口
- **第三节**：信息选择与压缩——关键点提取 / 结构化转换 / 公式化表达
- **第四节**：分块策略——五种分块方式、块大小选择、元数据
- **第五节**：上下文增强技术——RAG、记忆系统、混合检索
- **第六节**：两个完整实战案例——企业知识库 RAG + Agent 记忆系统
