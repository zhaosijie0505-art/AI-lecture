# 第五章：Agent 核心概念

![cover](image/agent-core-concepts-cover.png)

**LLM 是大脑，Agent 是会行动的智能体。**

前三章我们讨论了如何更好地提问（Prompt 工程）、如何给 LLM 提供更好的上下文（上下文工程）、以及如何量化 AI 系统的表现（Harness 工程）。但这些都是在"单次交互"的框架下讨论的——用户问一次，AI 答一次。

但真正强大的 AI 应用需要 AI 能够：
- **自主规划**：把一个复杂任务拆解成多个步骤
- **使用工具**：调用外部工具（搜索、计算、数据库查询）来完成任务
- **记住过程**：在多轮对话中保持上下文连贯
- **自我反思**：发现错误后纠正自己

这就是 **Agent（智能体）** 的概念——一个有目标、会推理、能行动、可自省的 AI 系统。

## 从 LLM 到 Agent：质的变化

### LLM 的局限

传统 LLM 的工作模式：

```
用户 → "北京天气如何？" → LLM → "北京今天晴天，25度"
```

LLM 的局限：
- **无法获取实时信息**：训练数据截止日期之后的知识它不知道
- **无法执行操作**：只能生成文本，不能真正"做事"
- **无法访问私有数据**：企业内部数据库、用户个人数据
- **复杂任务处理能力弱**：多步骤任务容易在中间步骤出错

### Agent 的工作模式

Agent = LLM + 工具 + 记忆 + 推理引擎：

```
用户 → "帮我订明天北京到上海的机票" → Agent
                                      ↓
                            ┌─────────────────┐
                            │  推理引擎(Reason)  │
                            │  1. 用户要订机票   │
                            │  2. 需要出发地/目的地/日期 │
                            │  3. 调用工具查询航班 │
                            └─────────────────┘
                                      ↓
                            ┌─────────────────┐
                            │  行动(Act)       │
                            │  search_flights  │
                            │  参数: Beijing-Shanghai-2024-12-25 │
                            └─────────────────┘
                                      ↓
                            ┌─────────────────┐
                            │  工具返回结果     │
                            │  3个航班可选      │
                            │  显示给用户确认   │
                            └─────────────────┘
                                      ↓
                            用户选择 → 调用 book_flight 工具 → 订票成功
```

## Agent 的四大核心能力

```
┌─────────────────────────────────────────────────────────────┐
│                    Agent 四大核心能力                       │
├─────────────────────────────────────────────────────────────┤

│   · Chain-of-Thought（思维链）                              │
│   · ReAct（推理+行动）                                      │
│   · Plan-and-Execute（规划+执行）                           │
│                                                             │
│   行动（Acting）                                            │
│   ━━━━━━━━━━━━━━━━━━━━━━━                                 │
```
