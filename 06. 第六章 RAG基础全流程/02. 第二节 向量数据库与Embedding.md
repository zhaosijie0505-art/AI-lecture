# 第二节 向量数据库与 Embedding

## 什么是向量数据库

向量数据库是专门存储和检索高维向量（embedding）的数据库。

```
传统数据库 vs 向量数据库：

传统数据库：
存储：结构化数据（表格、键值）
查询：精确匹配（WHERE id = 1）
适用：CRM、ERP 等业务系统

向量数据库：
存储：高维向量（1536维、3072维...）
查询：相似度搜索（找最相似的）
适用：语义搜索、推荐系统、RAG
```

## Embedding 原理

### 什么是 Embedding

Embedding 是把文本、图片等内容转换为固定维度向量的过程：

```
文本 → Embedding 模型 → 向量

"苹果" → [0.12, -0.34, 0.56, ..., 0.78]  # 1536 维向量
"水果" → [0.11, -0.31, 0.58, ..., 0.75]
"手机" → [0.89, 0.12, -0.23, ..., 0.34]

语义相似的文本，向量也相似：
"苹果" 和 "水果" 的距离 ≈ 0.15（很近）
"苹果" 和 "手机" 的距离 ≈ 2.31（很远）
```

### Embedding 模型的选择

| 模型 | 维度 | 特点 | 适用场景 |
|------|------|------|---------|
| **text-embedding-3-small** | 1536 | OpenAI 最新小型模型，便宜 | 通用场景 |
| **text-embedding-3-large** | 3072 | OpenAI 最新大型模型，效果好 | 高精度需求 |
| **text-embedding-ada-002** | 1536 | OpenAI 老牌模型，稳定 | 兼容性好 |
| **BGE-large-zh** | 1024 | 国产开源，支持中文 | 中文场景 |
| **m3e-large** | 1024 | 国产开源，中文效果好 | 中文场景 |

```python
# OpenAI Embedding 示例
from openai import OpenAI

client = OpenAI()

def get_embedding(text: str, model: str = "text-embedding-3-small") -> list:
    """获取文本的 embedding 向量"""
    response = client.embeddings.create(
        input=text,
        model=model
    )
    return response.data[0].embedding

# 示例
vec1 = get_embedding("机器学习")
vec2 = get_embedding("深度学习")
vec3 = get_embedding("做菜")

# 计算余弦相似度
from numpy import dot
from numpy.linalg import norm

def cosine_sim(a, b):
    return dot(a, b) / (norm(a) * norm(b))

print(f"机器学习 vs 深度学习: {cosine_sim(vec1, vec2):.3f}")  # ~0.85（相似）
print(f"机器学习 vs 做菜: {cosine_sim(vec1, vec3):.3f}")      # ~0.15（不相似）
```

### ChromaDB（开发首选）

```python
# ChromaDB 快速上手
import chromadb

# 创建客户端（本地模式）
client = chromadb.Client()

# 创建 Collection
collection = client.create_collection(
    name="my_knowledge_base",
    metadata={"description": "我的知识库"}
)

# 添加文档
collection.add(
    documents=[
        "机器学习是人工智能的一个分支",
        "深度学习使用神经网络进行学习",
        "Python 是一种编程语言"
    ],
    ids=["doc1", "doc2", "doc3"],
    embeddings=[
        get_embedding("机器学习是人工智能的一个分支"),
        get_embedding("深度学习使用神经网络进行学习"),
        get_embedding("Python 是一种编程语言")
    ]
)

# 查询
results = collection.query(
    query_embeddings=[get_embedding("什么是深度学习")],
    n_results=2
)

print(results["documents"])  # 返回最相关的 2 个文档
```

### FAISS（高性能场景）

```python
# FAISS 快速上手
import faiss
import numpy as np

# 准备数据（1000 个 768 维向量）
dimension = 768
num_vectors = 1000
vectors = np.random.random((num_vectors, dimension)).astype('float32')

# 创建索引
index = faiss.IndexFlatL2(dimension)  # L2 距离
index.add(vectors)

# 查询
query_vector = np.random.random((1, dimension)).astype('float32')
distances, indices = index.search(query_vector, k=5)  # 找最近的 5 个

print(f"最近邻索引: {indices}")
print(f"距离: {distances}")
```

## 相似度计算

### 余弦相似度（Cosine Similarity）

最常用的相似度度量，值范围 [-1, 1]：

```
cosine_sim(A, B) = (A · B) / (||A|| × ||B||)

值 = 1：完全相同
值 = 0：正交（无关）
值 = -1：完全相反
```

```python
def cosine_similarity(a: list, b: list) -> float:
    """计算两个向量的余弦相似度"""
    import math

    dot_product = sum(x * y for x, y in zip(a, b))
    magnitude_a = math.sqrt(sum(x ** 2 for x in a))
    magnitude_b = math.sqrt(sum(x ** 2 for x in b))

    if magnitude_a == 0 or magnitude_b == 0:
        return 0.0

    return dot_product / (magnitude_a * magnitude_b)
```

### 点积（Dot Product）

也叫余弦相似度的未归一化版本：

```python
def dot_product(a: list, b: list) -> float:
    """计算向量点积"""
    return sum(x * y for x, y in zip(a, b))

# 向量已归一化时，点积等价于余弦相似度
# 点积计算更快，适合大规模检索
```

### L2 距离（欧氏距离）

```python
import math

def euclidean_distance(a: list, b: list) -> float:
    """计算欧氏距离"""
    return math.sqrt(sum((x - y) ** 2 for x, y in zip(a, b)))

# 值越小越相似
```

### 选哪个

| 度量方式 | 适用场景 |
|---------|---------|
| **余弦相似度** | 语义相似度搜索（推荐，默认选择） |
| **点积** | 已归一化向量，高性能场景 |
| **欧氏距离** | 图像/音频特征匹配 |

### 1. 中文 embedding 模型

# 使用 BGE 中文 embedding
from sentence_transformers import SentenceTransformer

model = SentenceTransformer('BAAI/bge-large-zh-v1.5')

def embed_chinese(text: str) -> list:
    """中文文本 embedding"""
    return model.encode(text, normalize=True).tolist()

# 测试
vec1 = embed_chinese("人工智能将改变世界")

### 2. 批量 embedding

```python
# 批量处理提升效率
def embed_batch(texts: list, batch_size: int = 100) -> list:
    """批量生成 embedding"""
    all_embeddings = []

    for i in range(0, len(texts), batch_size):
        batch = texts[i:i + batch_size]
        # 调用 embedding API
        response = client.embeddings.create(
            input=batch,
            model="text-embedding-3-small"
        )
        embeddings = [item.embedding for item in response.data]
        all_embeddings.extend(embeddings)

        print(f"已处理 {min(i + batch_size, len(texts))}/{len(texts)}")

    return all_embeddings

# 10000 条文本，每批 100 条，100 次 API 调用
```

### 3. Embedding 质量评估

```python
def evaluate_embeddings(model, test_pairs: list) -> float:
    """
    评估 embedding 模型质量
    test_pairs: [(文本1, 文本2, 是否相似), ...]
    """
    correct = 0

    for text1, text2, expected_similar in test_pairs:
        vec1 = model.encode(text1)
        vec2 = model.encode(text2)
        similarity = cosine_similarity(vec1, vec2)

        predicted_similar = similarity > 0.7
        if predicted_similar == expected_similar:
            correct += 1

    return correct / len(test_pairs)

# 测试
test_pairs = [
    ("机器学习", "深度学习", True),
    ("机器学习", "烹饪技术", False),
    ("今天天气好", "天气很不错", True),
    ("苹果手机", "三星手机", True),
    ("苹果", "吃苹果", False),  # "苹果"水果 vs 苹果公司
]

accuracy = evaluate_embeddings(bge_model, test_pairs)
print(f"Embedding 准确率: {accuracy:.2%}")
```

## 本节小结

**Embedding** 是把文本转为向量的技术，语义相似的文本，向量也相近。

**向量数据库** 专门存储和检索向量，支持相似度搜索，是 RAG 的基础设施。

**主流向量数据库**：ChromaDB（开发首选）、FAISS（高性能）、Milvus（生产环境）、Pinecone（云服务）。

**相似度计算**：余弦相似度（最常用）、点积（高效）、欧氏距离（图像/音频）。

**中文 Embedding**：推荐使用 BGE-large-zh 或 m3e-large，效果远好于纯英文模型。
