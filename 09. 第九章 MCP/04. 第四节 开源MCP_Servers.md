# 第四节 开源 MCP Servers

## 官方 MCP Servers

Anthropic 官方维护的 MCP Server：

### 1. 文件系统 Server

```bash
# 安装
npx -y @modelcontextprotocol/server-filesystem /path/to/allowed/directory

# 功能
- read_file：读取文件
- write_file：写入文件
- list_directory：列出目录
- create_directory：创建目录
- move_file：移动文件
- search_files：搜索文件
```

### 2. GitHub Server

```bash
# 安装
npx -y @modelcontextprotocol/server-github

# 需要设置 GITHUB_PERSONAL_ACCESS_TOKEN 环境变量

# 功能
- search_repositories：搜索仓库
- get_file_contents：获取文件内容
- create_file：创建文件
- update_file：更新文件
- list_issues：列出 Issue
- create_issue：创建 Issue
- list_pull_requests：列出 PR
- create_pull_request：创建 PR
```

### 3. Slack Server

```bash
# 安装
npx -y @modelcontextprotocol/server-slack

# 环境变量
SLACK_BOT_TOKEN=xapp-xxx
SLACK_TEAM_ID=Txxx

# 功能
- channels_list：列出频道
- channels_history：获取频道消息
- chat_postMessage：发送消息
- reactions_add：添加反应
```

### 4. Brave 搜索 Server

```bash
# 安装
npx -y @modelcontextprotocol/server-brave-search

# 环境变量
BRAVE_API_KEY=xxx

# 功能
- brave_search：网页搜索
```

### 5. Google Maps Server

```bash
# 安装
npx -y @modelcontextprotocol/server-google-maps

# 环境变量
GOOGLE_MAPS_API_KEY=xxx

# 功能
- get_place_details：地点详情
- search_places：搜索地点
- directions：路线规划
```

## 社区 MCP Servers

### 1. 数据库相关

#### PostgreSQL MCP Server

```bash
npm install -g @joshjeong/mcp-postgres

# 配置
{
  "host": "localhost",
  "port": 5432,
  "database": "mydb",
  "user": "user",
  "password": "password"
}

# 功能
- execute_sql：执行 SQL 查询
- list_tables：列出表
- describe_table：表结构
```

#### SQLite MCP Server

```bash
pip install mcp-server-sqlite

# 功能
- query：执行 SQL
- tables_list：列出表
- table_info：表信息
```

## 在 Claude Desktop 中配置

### 1. 配置文件位置

- macOS：`~/Library/Application Support/Claude/claude_desktop_config.json`
- Windows：`%APPDATA%\Claude\claude_desktop_config.json`
- Linux：`~/.config/Claude/claude_desktop_config.json`

### 2. 配置示例

```json
{
  "mcpServers": {
    "filesystem": {
      "command": "npx",
      "args": ["-y", "@modelcontextprotocol/server-filesystem", "/Users/demo/projects"],
      "env": {}
    },
    "github": {
      "command": "npx",
      "args": ["-y", "@modelcontextprotocol/server-github"],
      "env": {
        "GITHUB_PERSONAL_ACCESS_TOKEN": "ghp_xxx"
      }
    },
    "brave-search": {
      "command": "npx",
      "args": ["-y", "@modelcontextprotocol/server-brave-search"],
      "env": {
        "BRAVE_API_KEY": "xxx"
      }
    },
    "my-custom-server": {
      "command": "python",
      "args": ["/Users/demo/mcp-servers/my-server/main.py"],
      "env": {
        "API_KEY": "xxx"
      }
    }
  }
}
```

### 3. 多 Server 配置

```json
{
  "mcpServers": {
    "dev-tools": {
      "command": "python",
      "args": ["/path/to/dev-tools-server/main.py"],
      "env": {}
    },
    "prod-tools": {
      "command": "python",
      "args": ["/path/to/prod-tools-server/main.py"],
      "env": {}
    }
  }
}
```

## Server 发现和使用

### 1. Awesome MCP 列表

GitHub 上有社区维护的 awesome-mcp-servers 列表：
- 100+ 现成 Server
- 按功能分类（数据库、云服务、消息等）
- 包含部署说明

### 2. Server 质量评估

```
评估 MCP Server 质量的维度：

1. 功能完整性
   - 覆盖了多少 API？
   - 是否支持所有主要操作？

2. 安全性
   - 是否正确处理敏感信息？
   - 是否有适当的权限控制？

3. 文档质量
   - 是否有清晰的使用说明？
   - 是否有示例代码？

4. 维护状态
   - 最近更新时间？
   - Issue 响应速度？

5. 兼容性
   - 是否兼容最新协议版本？
   - 跨平台支持？
```
