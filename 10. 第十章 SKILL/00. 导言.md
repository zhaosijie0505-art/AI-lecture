# 第十章：SKILL

![cover](image/skill-cover.png)

**让 Claude 学习你的工作方式，然后永远做对。**

你有没有遇到过这种情况：花 20 分钟写了一个超详细的 Prompt，结果 Claude 第二次用的时候输出格式又飘了；或者团队里每个人用 Claude 的方式都不一样，产出一致性几乎为零；或者你的最佳实践和流程规范只有你自己知道，换个人来做就是另一个质量。

这些问题，不是 Prompt 写得不够好，而是**缺少一种机制，把专业知识、流程规范和最佳实践封装成可复用的能力**。

这就是 Skills 要解决的核心问题。

## 什么是 Skills（先行预告）

Skills 是 Anthropic 在 2025-2026 年推出的核心能力，它的核心思想是：

> **把专业知识、流程步骤和最佳实践打包成可复用的"技能包"，Claude 在任何场景下都能自动调用。**

和之前我们学过的概念做一个快速对比：

| 概念 | 本质 | 触发方式 |
|------|------|----------|
| **Tool** | 模型调用的外部能力（搜索、代码执行等） | Agent 决策调用 |
| **Agent** | 能自主完成多步骤任务的智能体 | 人类给目标 |
| **RAG** | 从文档库检索相关知识注入上下文 | Query 触发检索 |
| **Skill** | 封装好的专业流程和最佳实践 | 场景自动匹配 |
