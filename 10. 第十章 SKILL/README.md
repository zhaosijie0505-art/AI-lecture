## 核心概念

**Skill = 把专业知识、流程规范和最佳实践打包成可复用能力单元**

Anthropic 官方定义：
> *Skills package your expertise and procedures so Claude delivers expert-level output every time.*

## 章节速览

- **第一节**：理解 Skill 的本质——它不是 Tool、不是 Agent、不是 RAG，是封装专业能力的新抽象
- **第二节**：掌握 Skills 的四大核心价值——一致性、知识封装、跨平台复用、组合使用
- **第三节**：通过 Figma、Box、Notion、Canva 等真实案例理解 Skills 的落地效果
- **第四节**：深入技术原理——Skill 的结构、注册、调用、Stack 组合
- **第五节**：手把手构建你的第一个 Skill，完整实践流程
