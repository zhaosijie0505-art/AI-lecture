# 第十四章：开源 Agent 项目解析 —— Hermes Agent

![cover](image/hermes-architecture-cover.png)

学完前面的 Agent 核心理论、RAG 工程、MCP 协议、SKILL 系统、CLI 开发，你已经具备了从零设计一个 Agent 系统的全部知识。但\"会造车\"和\"能造好车\"之间，还隔着一层——**看高手怎么造**。

本章选择 Hermes Agent 作为解剖对象。它是 Nous Research 出品的开源 AI Agent，目前在 GitHub 上有近万星，实现了真正产品级的多平台接入、持久化记忆、工具系统、子代理委托和定时任务。通过阅读它的源码，你将看到：教科书上的设计原则如何在 production 级别的代码中落地、模块边界怎么划分、坑怎么踩、演进历史怎么在代码里留下痕迹。

这也是本书唯一一章\"阅读他人代码\"的内容。我会带着你一行行分析核心模块，不是翻译代码，而是讲清楚**为什么这样设计**，以及**你能在自己的项目里学什么**。
