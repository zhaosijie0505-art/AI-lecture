# AI 学习课程

一门面向工程师的 AI 实战课程，从思维转型到落地工程，覆盖 Prompt、Agent、RAG、MCP 等核心主题。

## 课程目录

| 章节 | 主题 | 说明 |
|------|------|------|
| 第一章 | AI 思维 | 认知转型、概率思维、工程师角色转变、人机协作模型 |
| 第二章 | Prompt 工程 | 提示词设计与优化 |
| 第三章 | 上下文工程 | 上下文管理与优化 |
| 第四章 | Harness 工程 | AI 工作环境搭建 |
| 第五章 | Agent 核心概念 | Agent 架构与设计模式 |
| 第六章 | RAG 基础全流程 | 检索增强生成入门 |
| 第七章 | Agent 开发框架 | 主流框架对比与实践 |
| 第八章 | RAG 进阶 | 高级检索与优化策略 |
| 第九章 | MCP | Model Context Protocol |
| 第十章 | SKILL | 技能系统设计 |
| 第十一章 | CLI | 命令行工具开发 |
| 第十二章 | 项目实战 1 | AI 数学助教 |
| 第十三章 | 项目实战 2 | 公司办公助手 |
| 第十四章 | 开源 Agent 项目解析 | 典型项目源码阅读 |

## 文档规范

每章包含：
- `00. 导言.md` — 章节封面图 + 150-200 字导言
- `01~0N. 第X节 节名.md` — 各小节正文
- `image/` — 章节配图

## 适合人群

有一定编程基础（尤其是后端开发经验），想系统学习如何在工程实践中用好 AI 的开发者。
